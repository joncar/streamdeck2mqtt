# streamdeck2mqtt

## Requirements
```
Download Roboto Regular from https://fonts.google.com/specimen/Roboto
Download Material Design Icons font from https://github.com/Templarian/MaterialDesign-Webfont/archive/master.zip
pip3 install streamdeck pillow paho-mqtt
sudo apt-get install libhidapi-libusb0 libtiff5 libopenjp2-7
sudo cp udev-streamdeck.rules /etc/udev/rules.d/10-streamdeck.rules
```
